1. Basic structure of a C# program

A basic C# program is categorized in namespaces, classes and methods.

The following example shows the basic skeleton of a C#-program

namespace Introduction
{
      class program
      {
              static void Main()
              {

              }
      }
}

In the beginning you write all commands between the brackets of the "static void Main()"-method.


2. Basic output

If you want to print something on the terminal, for example "Hello World!", you use the commmand "System.Console.WriteLine("Hello World!")"

Everything that is written between those brackets will be printed.

You implement this so: 

namespace Introduction
{
      class program
      {
              static void Main()
              {
                      System.Console.WriteLine("Hello World!");
              }
      }
}

It´s important that you end a command with this symbol ";". So the program knows that the command is here finished.


3. Different Variable types and their usage

In programming there is something called Variabletypes, or Datatypes, with which you can declare the characteristic of a variable.

There are two types:

  • value types
  • reference types

We will discuss later, which one does what.

Value types

They are catagorized in three Sections:

  • Integer types
  • Floating-point types
  • Bool

Integer types

With integer types you can declare an integer value.

This is the list with the different integer types:

  • sbyte

    • A sbyte is a signed 8-bits integer. That means you can declare it with a number between -128 and 127.


  • byte

    • A byte is a unsigned 8-bits integer. That means you can declare it with a number between 0 and 255(2^8).


  • short

    • A short is a signed 16-bits integer. That means you can declare it with a number between -32,768 and 32,767.


  • ushort

    • A ushort is a unsigned 16-bits integer. That means you can declare it with a number between 0 and 65,535(2^16).


  • int

    • A int is a signed 32-bits integer. That means you can declare it with a number between -2,147,483,648 and 2,147,483,647.


  • uint

    • A uint is a unsigned 32-bits integer. That means you can declare it with a number between 0 and 4,294,967,295(2^32).


  • long

    • A long is a signed 64-bits integer. That means you can declare it with a number between -9,223,372,036,854,775,808 and 9,223,372,036,854,775,807.


  • ulong

    • A ulong is a unsigned 64-bits integer. That means you can declare it with a number between 0 and 18,446,744,073,709,551,615(2^64).


At the beginnning, we would recommend to use int.

Floating-point types

With floating-point types you can declare a floating-point number value.

This is the list with the different floating-point types:

  • float

    • A float is a signed 4-byte float. That means you can declare it with a number between ±1.5 x 10^-45 and ±3.4 x 10^38.


  • double

    • A double is a signed 8-byte float. That means you can declare it with a number between ±5.0 × 10^-324 and ±1.7 × 10^308.


  • decimal

    • A decimal is a signed 16-byte float. That means you can declare it with a number between ±1.0 x 10^-28 and ±7.9228 x 10^28.


Bool

A bool describes if a statement is true or false.


Declaration and Initialization

You can declare it like this:

int number1;
double number2;
bool isvalid;

After the Valuetype comes the variable-name


If you want to initialize it, you must give the variable-name and a value:

number1 = 5;
number2 = 1.5;
isvalid = true;

You can do both in one row:

int number1 = 5;
double number2 = 1.5;
bool isvalid = true;
